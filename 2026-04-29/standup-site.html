Meta

Built this standup site

This page you’re reading — plus the index, plus each detail page — is the thing. A standing location for daily standup reports so a single URL in Slack covers the headline and anyone who wants more can drill in.

Shape

  • Root index lists every day, newest first.
  • Each day gets its own folder with an index.html that holds the bullet list.
  • Each bullet links to a per-item detail page so the depth can grow without bloating the daily post.

Why

Slack standup posts get long or get sparse; neither is great. A link to a page with short bullets (each one a proper hyperlink) keeps the main post scannable while giving context for anyone who wants it. Yesterday’s live too — the index is chronological.

Hosting

Public repo on GitHub, served via GitHub Pages. Public but obscure — nobody stumbles on it, the link in Slack is how coworkers reach it. No secrets, no confidential code, just summaries of what shipped.
