What landed

  • Playwright + Electron added as dev dependencies.
  • Harness at tests/ui/, runnable via npm run test:ui.
  • Launches the prebuilt bundle (dist-electron/main.js) — same binary the DMG ships.
  • Bedrock mock: ADVISOR_MOCK_BEDROCK=1 short-circuits every AI-service call to deterministic stubs. No AWS creds, no network.

Coverage (initial)

  • Smoke per tab — does each one load?
  • Ambient strip renders with expected content.
  • Code-tab hidden-sidebar layout (regression guard for the fix from Tuesday).
  • Cmd+1 through Cmd+5 tab navigation.

Annotated failure screenshots

  • On fail: capture screenshot, overlay arrows/boxes/labels on the broken region using sharp.
  • Playwright report includes the annotated PNG, not raw stdout.
  • Result: you see what broke, not just that it broke.

Two gotchas documented

  • Mock env var read once at module load. Flipping at runtime has no effect — restart the process.
  • Parent-death SIGTRAP when launched from a non-Aqua shell (certain sudo contexts). Fix: wrap with launchctl asuser. Documented in docs/advisor-implementation-notes.md.

Decisions on the fork

  • Launch the prebuilt bundle, not npm run dev. Playwright’s _electron.launch handles bootstrap differently than spawning the dev server. Use the DMG-ish shape so tests reflect production.
  • Consequence: main-process edits require npx vite build before npm run test:ui.
