Six link types

  • Claude Code session (opens Code tab to that sid)
  • Slack-authored session (same, URL self-documents origin)
  • Plan file (opens Plans tab to that file)
  • Task (opens Tasks tab to that task)
  • Advisor day thread (opens Advisor to that day)
  • Project (opens Tasks, scrolls to the project’s group)

How it’s split

  • Receiver in Advisor: parses URL, routes to the right tab. Unknown URL → toast, no crash.
  • Sender in PittsBot: one helper every message builder uses. Raises on bad input; never emits an unknown type.
  • Spec: one doc, two repos, identical bytes. diff -q confirms.
