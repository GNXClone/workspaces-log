Item 2

Three Advisor UI polish fixes

Three small-but-visible items from the Advisor backlog, shipped on the same branch. New tests for each where logic changed.

1. Hidden-sidebar layout fix

When the sidebar was hidden, the Advisor’s top bar overlapped the ambient strip and the first line of tab content — visible misalignment at the top edge. Fix: the title bar is now a flex child that always takes 44px, so content reflows cleanly whether the sidebar is open or hidden. Also added a project / session breadcrumb to the top bar when the Code tab is active and the sidebar is hidden.

2. Sidebar-toggle hover tooltip

Small thing, long-standing irritation. Hovering the sidebar toggle now shows a tooltip with a state-aware label — Show sidebar when it’s hidden, Hide sidebar when it’s visible. New Tooltip primitive added to the design system; documented in the UI guidelines so future hover labels go through it instead of the browser’s default title="".

3. Manually renamed sessions stay renamed

When you rename a Claude Code session via the /rename command in the CLI, Advisor was overwriting the new name with an auto-generated one on the next cache refresh. Now: if a user-set name exists in the session log, Advisor uses it verbatim and skips the AI-generated title entirely. The cache distinguishes the two sources so downstream readers can tell which is which. Five new tests cover the precedence rules.

Verification

All three shipped as independent commits. TypeScript clean. Full unit-test suite green after each one.
