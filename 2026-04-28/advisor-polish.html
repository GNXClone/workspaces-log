1. Hidden-sidebar layout (CC-B-005)

  • Hiding sidebar left top-bar overlapping ambient strip + first line of content.
  • Title bar is now a flex child that always takes 44px; content reflows cleanly in both states.
  • Code tab also gets a project / session breadcrumb in the top bar when sidebar is hidden.

2. Sidebar-toggle hover tooltip (CC-B-006)

  • New Tooltip primitive added to the design system.
  • State-aware label: “Show sidebar” when hidden, “Hide sidebar” when visible.
  • Documented in the UI guidelines so future hover labels go through it, not browser title="".

3. Manual /rename wins over synth (CC-B-007)

  • When a session was renamed via /rename in the CLI, Advisor was overwriting the new name with an AI-generated one on the next refresh.
  • Now: if a user-set name exists in the session log, Advisor uses it verbatim and skips synthesis.
  • Cache entry carries a distinct source marker so downstream readers can tell “user-set” from “bedrock-sonnet.”
  • 5 new unit tests cover precedence + re-rename invalidation.
